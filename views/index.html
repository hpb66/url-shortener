API Basejump: URL shortener microservice

To get API:

Try inserting the following url on your browser:

Example Usage

https://same-flock.glitch.me/new/https://www.google.com

Creation output

{ "original_url":"http://foo.com:80", "short_url":"https://same-flock.glitch.me/8170" }

Will redirect to

https://www.google.com
